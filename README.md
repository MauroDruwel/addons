## üöÄ Getting Started

To add this repository to your Home Assistant instance and install add-ons, follow these steps:

### 1. Add the Repository

**Automatic:** Click the button below:

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fmaurodruwel%2Faddons)

**Manual:**
1. Open Home Assistant in your browser.
2. Go to **Settings** > **Add-ons** > **Add-on Store**.
3. Click the three dots in the top right corner and select **Repositories**.
4. Enter the following URL and click **Add**:

```
https://github.com/maurodruwel/addons
```

### 2. Install an Add-on

1. Find the add-on you want in the list.
2. Click on it, then click **Install**.
3. Configure the add-on options if needed, then click **Start**.
